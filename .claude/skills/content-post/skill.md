# Content Post Skill

유튜브 영상 URL 또는 메타데이터를 받아 리치인터뷰 포스트 HTML을 생성한다.

## 입력
- 유튜브 영상 URL 또는 VIDEO_ID
- 인물명 (한국어 + 영문)
- (선택) 트랜스크립트

## 실행 단계

### 1. 정보 수집
- 유튜브 영상 메타데이터 확인 (제목, 길이, 설명)
- 해당 인물의 기존 프로필 확인 (`people/` 디렉토리)
- 기존 포스트 중 최신 파일을 템플릿으로 읽기

### 2. 콘텐츠 생성
- 영상 분석 기사 작성 (3-5개 섹션: 핵심 메시지, 상세 분석, 시사점, 실천 포인트)
- HTML 파일 생성 (기존 템플릿 구조 정확히 따르기)
- 파일명: 한글 제목을 하이픈 연결 (예: `젠슨-황의-AI-인사이트-EP2.html`)

### 3. 사이트 연동
- `index.html` 포스트 그리드에 카드 추가 (최상단)
- 해당 인물의 `people/{이름}.html`에 에피소드 추가
- `sitemap.xml`에 새 URL 추가 (priority 0.8)

### 4. 검증
- HTML 구조가 기존 포스트와 일치하는지 확인
- 내부 링크 정합성 확인
- OG 태그, Schema.org LD+JSON 포함 확인
